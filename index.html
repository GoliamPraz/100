🌐 Учи HTML, CSS и JavaScript

Трите основни технологии за изграждане на уеб сайтове

Основни технологии

HTML

HyperText Markup Language — структурата на уеб страницата. Описва съдържанието чрез елементи и тагове като <h1>, <p>, <a> и <img>.

Структура

CSS

Cascading Style Sheets — визията на страницата. Управлява цветове, шрифтове, оформление и анимации, за да изглежда всичко красиво.

Стил

JavaScript

JavaScript — интерактивността на страницата. Реагира на действия на потребителя, обновява съдържанието и комуникира със сървъри.

Поведение

Интерактивна демонстрация

Натисни бутоните по-долу, за да видиш JavaScript в действие:

Натисни някой бутон!

Пример с код

<!-- HTML: структура -->
<button id="my-btn">Натисни ме</button>
<p id="msg"></p>

/* CSS: стил */
#my-btn {
  background: #1a73e8;
  color: white;
  padding: 10px 20px;
  border-radius: 8px;
}

// JavaScript: поведение
const btn = document.getElementById('my-btn');
btn.addEventListener('click', function () {
  document.getElementById('msg').textContent = 'Здравей, свят!';
});
