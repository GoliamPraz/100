Кратки знаци (оператори) — с пример и обяснение

const equal = (a === b);
=== (строго равенство)
Сравнява и стойност, и тип. Във файл като tt.js е добре за проверки в if, защото избягва неочаквани преобразувания на типове (например 5 и '5' не са равни).
if (status !== 'ok') { console.log('Грешка'); }
!== (строго различно)
Проверява дали стойностите са различни, без автоматично преобразуване. Полезно е при валидации на входни данни от prompt, форма или API.
if (isLoggedIn && isAdmin) { showPanel(); }
&& (логическо И)
Кодът в блока се изпълнява само ако и двете условия са истина. В реален файл се ползва за комбинирани проверки преди действие.
if (isMobile || isTablet) { compactMenu(); }
|| (логическо ИЛИ)
Достатъчно е едно условие да е истина. Подходящо е за fallback логика и избор между няколко валидни варианта.
const isHidden = !isVisible;
! (отрицание)
Обръща булевата стойност. Често се ползва за toggle поведение, както при бутон isPressed = !isPressed.
const text = isPressed ? 'Натиснат' : 'Натисни ме';
?: (тернарен оператор)
Кратък вариант на if/else, когато трябва да избереш една от две стойности. Много удобен за текст, цвят, клас и други бързи UI промени.
canvas.addEventListener('click', (event) => { console.log(event.type); });
=> (arrow function)
Кратък синтаксис за функция, особено удобен за callback-и. Пише се често в събития, map, filter, then и други места с малки функции.
const all = [...baseItems, newItem];
... (spread)
Разгъва масив/обект в нова структура. Ползва се за копиране и добавяне на елементи без директна мутация на оригинала.
function logAll(...args) { console.log(args); }
... (rest)
Събира много аргументи в един масив. Удобно е за функции, които приемат различен брой параметри.
const city = user?.address?.city;
?. (optional chaining)
Проверява безопасно вложени свойства. Ако някое ниво липсва, връща undefined, вместо да хвърли грешка и да спре изпълнението на файла.
const username = inputName ?? 'Гост';
?? (nullish coalescing)
Задава стойност по подразбиране само ако лявата страна е null или undefined. Това е по-точно от ||, когато '' или 0 са валидни стойности.

Условия и цикли

if (a > b) { /*...*/ } else { /*...*/ }
Условен клон за контрол на изпълнението.
switch(value) { case 1: break; }
Многократни условия — алтернатива на if/else ако има много стойности.
for (let i = 0; i < n; i++) { /*...*/ }
Класически for цикъл.
while (cond) { /*...*/ }
Цикъл с предусловие.
do { /*...*/ } while (cond);
Цикъл, който се изпълнява поне веднъж.

Функции

function add(a, b) { return a + b; }
Обявяване на функция и връщане на стойност.
const add = (a, b) => a + b;
Arrow функция — кратък синтаксис и лексически this.
return value;
Връща стойност от функция.
function fn(...args) { /* args е масив */ }
Rest параметри — събират останалите аргументи в масив.

Масиви

const arr = [1,2,3];
Създава масив.
arr.push(4); arr.pop(); arr.shift(); arr.unshift(0);
Добавяне/премахване от край или начало (mutating).
arr.forEach(x => console.log(x));
Итерация през масив — няма връщане на нов масив.
const mapped = arr.map(x => x*2);
map връща нов масив след трансформация.
const filtered = arr.filter(x => x%2===0);
filter връща масив с елементи, които преминават тест.
const sum = arr.reduce((acc,v) => acc+v, 0);
reduce агрегира стойности в една (напр. сума).
arr.find(x => x>2); arr.findIndex(x => x>2); arr.includes(3); arr.indexOf(3);
Търсене на елементи и проверки за наличие.
arr.slice(1,3); arr.splice(1,2,9,10); arr.concat([4,5]); arr.join(',');
Вадене на части, мутации, съединяване и превръщане в стринг.
Array.from(nodeList)
Преобразува iterable (напр. NodeList) в истински масив.

Обекти

const obj = { name: 'Ivo', age: 30 };
Създаване на обект с ключове и стойности.
obj.name // или obj['name']
Достъп до свойства.
const {name, age} = obj; const [a,b]=arr;
Деструктуриране на обекти и масиви.
const copy = {...obj}; const merged = {...a, ...b};
Spread оператор за копиране/сливане (shallow).
Object.keys(obj); Object.values(obj); Object.entries(obj);
Получаване на ключове, стойности или двойки [ключ,стойност].
JSON.stringify(obj); JSON.parse(jsonStr);
Сериализация и десериализация към/от JSON.

`Hello ${name}`
Template literals — интерполация в стринг.
const newArr = [...arr, 4];
Spread за масив — копиране и вмъкване.
const id = Symbol('id'); const big = 12345678901234567890n;
Symbol и BigInt — специфични нови типове.
class Player { constructor(name){this.name=name;} levelUp(){this.level++;} }
Класове (OOP синтаксис).
const copy = Object.fromEntries(Object.entries(obj).map(...))
Трикове с Object.entries/fromEntries за трансформации.
